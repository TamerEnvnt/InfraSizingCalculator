How K8s Resource Requests Work
Requests are the guaranteed resources for scheduling. Pods are placed on nodes that have enough allocatable capacity to satisfy requests.

Limits are the maximum resources a pod can burst to. CPU is throttled; memory exceeding limits triggers OOM kill.

Best Practice: Set memory request = limit for stability.

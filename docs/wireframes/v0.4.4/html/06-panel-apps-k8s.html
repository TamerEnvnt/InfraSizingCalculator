Resource Requests & Limits

K8s-Only Concept Resource requests/limits are Kubernetes concepts for QoS. Does NOT exist in VM mode.
Guaranteed: requests = limits (no overcommit)
Burstable: limits > requests (allows bursting)
QoS Classes (Kubernetes)
  • Guaranteed: Won't be evicted unless OOM
  • Burstable: Can use more if available
  • BestEffort: First to be evicted
Reference: [K8S-CLOUD] Resource QoS

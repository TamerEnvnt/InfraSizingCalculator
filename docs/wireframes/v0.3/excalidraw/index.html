Progressive Accordion Wireframes

Excalidraw format wireframes using the same design system as the HTML version. Uses normal font family (fontFamily: 2) for clean, professional appearance.

How to View These Wireframes

These .excalidraw files can be opened in:

1. Excalidraw Web App - Go to excalidraw.com and drag the file onto the canvas

2. VS Code - Install the "Excalidraw" extension and open files directly

3. Obsidian - Use the Excalidraw plugin to view and edit

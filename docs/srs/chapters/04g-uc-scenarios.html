UC-037

Save Scenario

Overview
Brief DescriptionAllows user to save current configuration and sizing results as a named scenario for future reference or comparison.
PriorityP2 (High)
ComplexityMedium
Applicable Business Rules
RulesBR-V008
Main Success Scenario
1
Actor: Clicks "Save Scenario" button
2
System: Displays save scenario modal
Fields:
- Scenario name (required)
- Description (optional)
- Tags (optional)
3
Actor: Enters scenario name and optional details
4
System: Validates name (1-100 characters, unique)
5
System: Saves scenario to database
6
System: Shows success confirmation

ComplexityMedium

Compare Scenarios

Overview
Brief DescriptionAllows user to select multiple scenarios and compare their configurations, sizing results, and costs side-by-side.
PriorityP2 (High)
ComplexityHigh
Main Success Scenario
1
Actor: Selects 2-4 scenarios for comparison
2
Actor: Clicks "Compare" button
3
System: Displays side-by-side comparison
Comparison Shows:
- Configuration differences (highlighted)
- Sizing differences
- Cost differences
- Percentage variations

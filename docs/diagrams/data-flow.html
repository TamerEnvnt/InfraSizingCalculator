Data Flow Diagram

Complete data flow from user input to export for Kubernetes and VM sizing calculations

Kubernetes Sizing Flow
1
User Input
Wizard Interface
User provides sizing parameters through the wizard:
Distribution (11 options)
Technology (6 options)
Cluster Mode
Applications per Tier
2
Build Input Model
K8sSizingInput
Construct the input object with all parameters:
DistributionConfig
TechnologyConfig + TierSpecs
AppConfig per Environment
NodeSpecs overrides
3
Calculate Sizing
K8sSizingService
Apply business rules to calculate node requirements:
BR-M001-M004: Master nodes
BR-I001-I006: Infra nodes
BR-W001-W006: Worker nodes
BR-H001-H003: Headroom
4
Generate Result
K8sSizingResult
Build comprehensive result object:
EnvironmentResult per env
Node counts & specs
Total resources (CPU/RAM)
Warnings & validations
5
Display & Export
ExportService
Present results and enable export:
Summary cards
Detailed tables
CSV / JSON / Excel
HTML Diagram
Virtual Machine Sizing Flow
1
User Input
Wizard Interface
User provides VM sizing parameters:
Technology (5 + OutSystems)
Environment(s)
Server roles per env
HA pattern selection
2
Build Input Model
VMSizingInput
Construct the VM input object:
TechnologyConfig + VMRoles
EnvironmentConfig[]
Role counts per env
Spec overrides
3
Calculate Sizing
VMSizingService
Apply technology-specific rules:
Standard roles (App, DB, Web)
OutSystems roles
HA pattern (Active-Passive/Active)
MaxInstances constraints
4
Generate Result
VMSizingResult
Build VM sizing result:
ServerResult per role
Per-environment breakdown
Total VM count & specs
Constraint warnings
5
Display & Export
ExportService
Present VM results and enable export:
VM summary cards
Role breakdown tables
Export formats
Key Business Rules Applied
BR-M001
Managed K8s (EKS, AKS, GKE, OKE) = 0 master nodes
BR-M002
Self-managed with HA = 3 master nodes
BR-I001
OpenShift only gets infrastructure nodes
BR-W003
Min replicas: Prod=3, Stage=2, Others=1
BR-H003-07
Default headroom: Prod/DR=37.5%, Dev/Test=33%, Stage=0%
VM-001
OutSystems Controller MaxInstances = 1

UC-025

Select Pricing Provider

Overview
Brief DescriptionAllows user to select the pricing source - On-Premises or a cloud provider (AWS, Azure, GCP, Oracle Cloud, IBM Cloud). This determines which pricing tables are used for cost estimation.
PriorityP1 (Critical)
ComplexityLow
Main Success Scenario
1
System: Displays pricing provider options
Options:
- On-Premises
- AWS
- Microsoft Azure
- Google Cloud Platform
- Oracle Cloud Infrastructure
- IBM Cloud
2
Actor: Selects provider
3
System: Loads provider-specific pricing tables and options

UC-027

Select Pricing Type

Overview
Brief DescriptionAllows user to select pricing model - Pay As You Go (on-demand), 1-Year Reserved, or 3-Year Reserved. Reserved instances provide significant discounts.
PriorityP1 (Critical)
ComplexityLow
Applicable Business Rules
RulesBR-P003
Main Success Scenario
1
System: Displays pricing type options
Options:
- Pay As You Go (PAYG / On-Demand)
- 1-Year Reserved (30-40% savings)
- 3-Year Reserved (50-60% savings)
2
Actor: Selects pricing type
3
System: Recalculates costs with selected pricing model
Traceability
Related RequirementsREQ-FR-027
ComponentsPricingTypeSelector.razor
UC-032

Configure On-Premises Pricing

Overview
Brief DescriptionAllows configuration of on-premises cost factors including hardware depreciation, power, cooling, datacenter space, and maintenance costs.
PriorityP1 (Critical) - For on-prem deployments
ComplexityMedium
Applicable Business Rules
RulesBR-P001
Main Success Scenario
1
System: Displays on-premises cost configuration
Configurable Factors:
- Hardware cost per server
- Depreciation period (years)
- Power cost ($/kWh)
- Cooling factor
- Datacenter space cost
- Annual maintenance percentage
Traceability

Wireframe Prototypes - No-Scroll, Split-Layout Design

Design Principles (v0.4.4)

No Scrolling

Every page fits within the viewport. Fixed height containers with overflow hidden.

Split Layout

Configuration panel visible alongside main content during the entire configuration journey.

Embedded Panels

Panels are part of dashboards (not overlays). Results and config visible simultaneously.

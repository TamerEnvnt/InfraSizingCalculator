Hover Effect Guidelines
  • Transitions: 0.2s ease for most interactions
  • Transform scale: 0.95-1.05 for subtle feedback
  • Focus rings: 3px blur for keyboard accessibility
  • Color shifts: Lighten on hover, darken on active
  • Reveal animations: Fade + translate for smoothness

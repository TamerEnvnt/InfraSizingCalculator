UC-017

View Summary Cards

Overview
Brief DescriptionDisplays high-level summary cards showing grand totals for nodes, CPU, RAM, and disk across all environments. Provides at-a-glance infrastructure overview.
PriorityP1 (Critical)
ComplexityLow
Main Success Scenario
1
System: Displays summary cards after calculation
Cards Displayed:
- Total Nodes (with breakdown by type)
- Total vCPU cores
- Total RAM (GB)
- Total Disk (GB)
- Estimated Monthly Cost (if pricing configured)
2
Actor: Views aggregated totals
3
Actor: Optionally clicks card to expand details

UC-018

View Detailed Sizing Results

Overview
Brief DescriptionDisplays detailed per-environment breakdown of sizing results including node counts, resource allocations, and configuration details in tabular format.
PriorityP1 (Critical)
ComplexityMedium
Main Success Scenario
1
System: Displays results table
Table Columns:
- Environment name
- Master nodes (count, CPU, RAM, Disk)
- Infra nodes (if OpenShift)
- Worker nodes (count, CPU, RAM, Disk)
- Environment totals
2
Actor: Reviews per-environment details
3
Actor: Optionally expands environment rows for more detail

UC-019

View Warnings and Recommendations

Overview
Brief DescriptionDisplays context-aware warnings and recommendations based on sizing results. Alerts user to potential issues like insufficient resources, over-provisioning, or configuration concerns.
PriorityP2 (High)
ComplexityMedium
Main Success Scenario
1
System: Analyzes sizing results for potential issues
2
System: Displays warning cards if issues found
Warning Types:
- High utilization risk (low headroom)
- Large cluster warning (100+ workers)
- HA recommendations
- Cost optimization suggestions
3
Actor: Reviews and acknowledges warnings
Traceability
Related RequirementsREQ-FR-019
ComponentsResultsWarnings.razor
